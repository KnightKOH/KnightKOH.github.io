AcWing 91.最短Hamilton路径（状压DP）

题目

​ 给定一张 nn 个点的带权无向图，点从 0~n-1 标号，求起点 0 到终点 n-1 的最短Hamilton路径。 Hamilton路径的定义是从 0 到 n-1 不重不漏地经过每个点恰好一次。

分析

​ 最短Hamilton路径，是一道NP完全问题，类似的有旅行商问题，周游各地求出不重不漏经过每点的最短路径。

​ 可以用dfs暴力处理，但数据量极大难以解决，所以考虑状态压缩。

举例：

假如有4个点 0 1 2 3

考虑这两种走法：0 -> 1 -> 2 -> 3 weight = 20

​ 0 -> 2 -> 1 -> 3 weight = 18

很显然我们会选第二种

​ 上面的例子可以引申出一些结论：对于每一点，我们只关心他是否被使用过，以及停在了哪点，而不关心它的使用顺序如何。

关键: 1. 哪些点被使用过。

​ 2, 最后停在哪点。

​ 到这里就有了点 dp 的感觉，接下来思考问题的总状态量：一共有 n 个点，每个点有两种状态（用或不用） n 的平方，所以总状态量是 n^2*n。

​ 而状态的标识正好与二进制中0和1的情况相符，所以考虑使用20位二进制数来表示状态：

还是四个点 0 1 2 3

若使用 0 1 2 ———> 0111

若使用 0 3 1 ———> 1011

​ 这就是状态压缩。

状态转移表达式： state 表示那些状态被使用 j 表示停在哪里 state_k 除去j之后的集合。

dp[state][j] = dp[state_k][k] + weight[k][j]

代码

#include<bits/stdc++.h>
using namespace std;
const int INF = 0x3f3f3f3f;
int dp[1<<20][25];
int weight[25][25];
int main() {
    ios::sync_with_stdio(false);
    memset(dp, INF, sizeof(dp));
    int n; cin >> n;
    for(int i = 0; i < n; i++)
        for(int j = 0; j < n; j++)
            cin >> weight[i][j];
    dp[1][0] = 0;    //第一个状态即起点，本身权值为零
    for(int i = 0; i < (1<<n); i++)
        for(int j = 0; j < n; j++)
            if(i >> j & 1)    //判断状态是否合法
                for(int k = 0; k < n; k++)    //枚举所有可用点
                    if(i - (1 << j) >> k & 1)    //减去j判断k
                        dp[i][j] = min(dp[i][j], dp[i-(1<<j)][k]+weight[k][j]);
    cout << dp[(1<<n)-1][n-1] << '\n';    //表示已经遍历了所有状态最后停在n-1点
    return 0;
}

技巧

​ 在状压DP中，常使用位运算表示集合，本题中判断状态是否合法时，就使用了位运算：判断一个数i的第j位（用二进制想）是否为1，将i右移j位 i >> j & 1
